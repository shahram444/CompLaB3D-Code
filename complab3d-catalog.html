Pore-Scale Reactive Transport Simulation

CompLaB3D is an open-source Lattice Boltzmann framework for simulating coupled fluid flow, solute transport, and geochemical reactions within 3D pore geometries derived from micro-CT scans or synthetic media.

Overview

What CompLaB3D Does

Simulate reactive transport at the scale where it actually happens ‚Äî inside the pore spaces of rocks, sediments, and porous materials.

üíß

Fluid Flow

Solve Navier-Stokes equations in complex pore geometries using the D3Q19 Lattice Boltzmann method. Calculate permeability, velocity fields, and identify preferential flow paths.

üåä

Solute Transport

Track how dissolved species move through pore spaces via advection and diffusion. Compute breakthrough curves, residence time distributions, and mixing behavior.

‚öóÔ∏è

Geochemical Reactions

Couple transport with chemical reactions ‚Äî from simple first-order kinetics to microbially-mediated processes like sulfate reduction and methane oxidation.

The Physics Inside

  • üî∑

    D3Q19 Lattice for Flow

    19 velocity directions capture incompressible Navier-Stokes behavior with BGK collision

  • üî∂

    D3Q7 Lattice for Transport

    Advection-diffusion equation solved with 7 velocity directions; operator splitting for reactions

  • üß±

    Bounce-Back Boundaries

    No-slip conditions at solid surfaces; pressure or velocity boundaries at inlet/outlet

  • ‚ö°

    MPI Parallelization

    Domain decomposition enables large-scale simulations across multiple processors

Getting Started

Installation

üìã Requirements

CompLaB3D requires a C++ compiler with MPI support:

  • C++11 compatible compiler (GCC 4.8+)
  • MPI library (OpenMPI or MPICH)
  • CMake 3.10+ (optional)
  • ParaView for visualization
üíª Build from Source
# Clone repository git clone https://github.com/your-repo/complab3d.git cd complab3d # Build mkdir build && cd build cmake .. make -j4
üöÄ Quick Start
# Run simulation (single processor) ./complab3d config.xml # Run with MPI (4 processors) mpirun -np 4 ./complab3d config.xml # Visualize results paraview output/concentration_*.vti
Reference

Technical Specifications

Component Method Details
Flow Solver D3Q19 Lattice Boltzmann BGK collision, incompressible limit
Transport Solver D3Q7 Lattice Boltzmann Advection-diffusion, operator splitting
Boundary Conditions Bounce-back, Zou-He No-slip walls, pressure/velocity BC
Reactions First-order, Monod kinetics Configurable rates, multi-species
Parallelization MPI 3D domain decomposition
Input Format Voxel data .raw .dat .vti
Output Format VTK ImageData .vti for ParaView
Configuration XML Human-readable parameter files
